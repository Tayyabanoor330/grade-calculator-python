# Student Grade Analyzer (Python)

This is a beginner Python project that analyzes student marks and calculates the final result.

## Features
- Accepts marks for multiple subjects
- Calculates total marks
- Calculates percentage
- Assigns a grade based on percentage
- Detects subjects where the student failed

## Technologies Used
- Python


## Example Output

Enter English marks: 80  
Enter Hindi marks: 70  
Enter Math marks: 90  
Enter Physics marks: 75  
Enter Chemistry marks: 60  

Total Marks: 375  
Percentage: 75.0%  
Final Grade: B+
